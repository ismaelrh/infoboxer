Wikimedia Infoboxer simulator

Introduction


You have to complete a half-filled provided Wikimedia Infobox code with the information about the person "{{ctrl.person}}".
As you see, on the code there are 'labelN' and 'dataN' keys. Each label specifies the name of the property to be described (for example 'Birth place') and its data (the one with the same number) contains the value of that property (for example New York).

The value can be:
  • A plain text value. E.g. New York
  • A reference to a Wikipedia page. E.g. [[ https://en.wikipedia.org/wiki/New_York ]] ( surrounded by "[[" and "]]").

What to do


In this hit you have to replace as many INSERT_TEXT values as you can with the correct value for the person within {{ctrl.availableMinutes}} minutes. You can search on Wikipedia or wherever you want.

Please try to insert references to Wikipedia pages whenever possible. If a Wikipedia page for that data doesn't exist, then insert a plain text value.

To do so:
  1. Enter a username and press Start when you are ready.
  2. Replace INSERT_TEXT with the appropriate values.
  3. When the time is about {{ctrl.availableMinutes}} minutes, please press the "Submit" button.
  4. Go back to Mechanical Turk page and insert the username you used.
