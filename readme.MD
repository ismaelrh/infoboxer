#HDT Infoboxer Prototype

##Hdt-sparql-endpoint-launcher-sources
Sources of the Sparql HDT Endpoint launcher.

##Hdt-sparql-endpoint-launcher-binary
Launches the Sparql HDT Endpoint. Please review "endpoint.[sh|bat]" and change it to fit your file locations.

##backend
Infoboxer backend, written in Java using Spring framework. Serves a REST api used to do operations querying the sparql-endpoint.
Also serves the frontend, available a src/main/resources/static
